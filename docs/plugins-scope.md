# Plugins Scope (audio.nmth.gov.tw)

原則：
- 只納管「自製/客製」外掛（我們可控、可維護、可回填）
- 第三方外掛整包不納管（wp.org/商業外掛可重裝；如有客製改動，改以 patch/紀錄處理）
- mxp-dev-tools 明確不納管
